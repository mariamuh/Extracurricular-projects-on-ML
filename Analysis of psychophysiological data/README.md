В рамках данного проекта я провела анализ данных, собранных с помощью психофизиологических методик (опросников), и рассмотрела метод случайного леса для получения прогнозов о том, как испытуемые относятся к неопределенности (см. темную триаду).

Таким образом, мне удалось: 
- обработать табличные данные о результатах прохождения опросников участниками испытания, создать на их основе признаки, необходимые для обучения и тестирования моделей,
- построить модель Random Forest.

В рамках исследования я изучала разные методы улучшения результатов полученной модели, включая использование всех/некоторых предикторов (признаков), установку различных значений гиперпараметров, перевод целевой переменной в альтернативную шкалу.
